A fancy menu built by AngularJS 2

Config Menu Gutter

The space between the menu and the boundaries of the page window. The object can have four properties left, right, top, bottom with values to be defined in Number. The measure unit is 'px'.

Default Value: top:{{defaultGutter.top}}px, right:{{defaultGutter.right}}px, bottom:{{defaultGutter.bottom}}px, left:{{defaultGutter.left}}px

Config Menu Position

Set the menu position when the page first loaded. Tips: you could drag the menu to different position

Default: {{defaultOptions.defaultPosition}}

Config Menu Font Family

The font family type of the menu.

Default: {{defaultOptions.font}}

Config Menu Default Open State

Config if open menu automatically on load.

Default: {{defaultOptions.defaultOpen}}

Config Menu Radius

The radius of the menu wings from the center of the button. The radius unit is 'px'.

Default: {{defaultOptions.radius}}px

Config Menu Spin State

Config if the menu could be span

Default: {{defaultOptions.spinable}}(could not be span)

Config Menu Show Icon State

Config if the menu shows icons

Default: {{defaultOptions.showIcons}}(show wing's icons)

Config Menu Only Icon State

Config if the menu shows only icons (Not shows wing title)

Default: {{defaultOptions.onlyIcons}}(show only wing's icons, no title)

Config Button offset

The gap between the menu button and the menu item wings. The width unit is 'px'.

Default: {{defaultOptions.offset}}

Config Button Background Color

Set the button background color.

Default: {{defaultOptions.buttonBackgroundColor}}

Config Button Width

Set the button width. The width unit is 'px'.

Default: 60px

Config Button Font and Cross Image Color

Set the button font color and the cross image color.

Default: {{defaultOptions.buttonFontColor}}

Config Button Font Size

Set the button font size. The font size unit is 'px'.

Default: 14px

Config Button Font Weight

Set the button font wight.

Default: 700

Config Button Opacity

Set the button opacity value when the button is not focused or open. Value could be set between 0~1.

Default: 0.7

Config Button Cross Image Size

Set the button cross image size. The size could be percentage('50%') or px('30px') or other measure units. Warning: do not forget to provide the measure unit(px, em, rem, %)

Default: 50%

Config Wing Angle

Set the angle at which each wing will open.

Default: {{defaultOptions.angle}} degrees

Config Menu Start Angles

The angle at which the menu wings open when clicked. The measure unit is 'deg'. The value could be minus(like: -30). For example, when you set the topLeft position start angle as '0'(default), the menu wings will open at "3 o'clock" degree before the menu expands.(Try it at the example menu.)

Default Value: Top Left: 0degree(3 o'clock), Top Right: 90degree(6 o'clock), Bottom Right: 180degree (9 o'clock), Bottom Left: 270degree (12 o'clock)

Config Wing General Font Size

Set the general font wing size. It controls all the wings' font size. The measure unit is 'px'

Default: {{defaultOptions.wingFontSize}}px

Config Menu General Font Weight

Set the general menu font wight.

Default: {{defaultOptions.wingFontWeight}}

Config Wing General Font and Icon Color

Set the general wing's font color and the icon color.(They can be overrode when you config every single wing)

Default: {{defaultOptions.wingFontColor}}

Config Menu Wing General Icon Size

Set the wing icon size. The measure unit is 'px'. (It can be overrode when you config every single wing)

Default: {{defaultOptions.wingIconSize}}px
