Config Menu Font Family

The font family type of the menu.

Default: {{defaultOptions.font}}

Config Menu Default Open State

Config if open menu automatically on load.

Default: {{defaultOptions.defaultOpen}}

Config Menu Radius

The radius of the menu wings from the center of the button. The radius unit is 'px'.

Default: {{defaultOptions.radius}}px

Config Menu Spin State

Config if the menu could be span

Default: {{defaultOptions.spinable}}(could not be span)

Config Menu Show Icon State

Config if the menu shows icons

Default: {{defaultOptions.showIcons}}(show wing's icons)

Config Menu Only Icon State

Config if the menu shows only icons (Not shows wing title)

Default: {{defaultOptions.onlyIcons}}(show only wing's icons, no title)

Config Button offset

The gap between the menu button and the menu item wings. The width unit is 'px'.

Default: {{defaultOptions.offset}}

Config Button Background Color

Set the button background color.

Default: {{defaultOptions.buttonBackgroundColor}}

Config Button Width

Set the button width. The width unit is 'px'.

Default: 60px

Config Button Font and Cross Image Color

Set the button font color and the cross image color.

Default: {{defaultOptions.buttonFontColor}}

Config Button Font Size

Set the button font size. The font size unit is 'px'.

Default: 14px

Config Button Font Weight

Set the button font wight.

Default: 700

Config Button Opacity

Set the button opacity value when the button is not focused or open. Value could be set between 0~1.

Default: 0.7

Config Button Cross Image Size

Set the button cross image size. The size could be percentage('50%') or px('30px') or other measure units. Warning: do not forget to provide the measure unit(px, em, rem, %)

Default: 50%

Config Wing Angle

Set the angle at which each wing will open.

Default: {{defaultOptions.angle}} degrees
